Welcome to IDTxl’s documentation!¶

Welcome to the IDTxl user documentation. IDTxl is being hosted at GitHub.

Have a look at the wiki for more information and user tutorials. If you want to report a problem, head to the issue tracker or have a look at the IDTxl google discussion group.
